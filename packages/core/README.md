# @next-cors/core

nextjs-cors's core

## Installation

> First we will have to install, in order to use this wonderful package.

```bash
# Using npm
npm install @next-cors/core

# Using yarn
yarn add @next-cors/core

# Using pnpm
pnpm add @next-cors/core
```
