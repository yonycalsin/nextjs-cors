# @next-cors/next-connect

next-cors for next-connect

## Installation

> First we will have to install, in order to use this wonderful package.

```bash
# Using npm
npm install @next-cors/next-connect

# Using yarn
yarn add @next-cors/next-connect

# Using pnpm
pnpm add @next-cors/next-connect
```
